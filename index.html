My Story

I graduated from the University of San Carlos as a scholar with a specialized 3 year course - Certificate in Systems and Network Administration, a joint project of the university and a French NGO Passerelles numériques

I helped managed the network as well as a few servers and support employees on their technical needs during my internship at AWS (Advanced World Systems).

Afted graduation, I got accepted in to US-based web consulting company where I started handling small to medium scale web hosting servers. These includes servers running applications common in a web hosting environment: Apache/NginX, PHP, MySQL, Redis/Memcache, NodeJS, etc. all running mostly on Linux Operating Systems.

Then I got into AWS(Amazon Web Services) and started migrating most of our applications from other hosting providers into AWS. I mostly use EC2, RDS, ElastiCache, S3 and CloudFront as well as Auto-Scaling and Load Balancing at the beginning.

And then I started working with containers and helped clients migrate their application into a containerized one to AWS ECS. These includes designing the clients' server infrastructure using CloudFormation.

All throughout these years I've used quite a number of tools and practices to help simplify my tasks. Technology is constantly evolving so I will keep updated with it by learning and exploring more tools and practices that will benifit me and my clients in whatever scenario we're at.
